## How It Works

This is a client-side only application that uses browser local storage to persist messages. Messages are stored locally in your browser and will be available when you return to the page.

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `script.js` - Message handling and local storage logic

## Usage

Simply visit the website, type your message in the textarea, and click "Send Message" (or press Enter). Your messages will be displayed below and saved in your browser.
